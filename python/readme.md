# Python 工具脚本 [WIP]

本目录计划放置与《火影忍者 RPG》汉化相关的 Python 脚本，与仓库其他模块配合使用。

## 计划功能

- **字模**：从 TTF 字体提取字模并转换为游戏使用的 8×8 / 8×16 等格式
- **文本**：从 ROM 导出剧情/菜单等文本，或把汉化文本写回 ROM
- **Binary diff**：生成「原版 ROM → 汉化 ROM」的 binary diff，供 [patcher](../patcher/) 使用；可选校验、合并等
